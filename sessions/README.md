# Sessions

This top-level folder holds session continuity documents for AI-assisted work on the
SDC.Schema solution: handoffs, kickstart prompts, and work summaries meant to let a new session
resume prior work. It is intentionally kept outside `SDC.Schema/docs/`, since these documents
describe *how* and *when* work was done rather than the solution's architecture.

Existing session documents currently live under
`SDC.Schema.Tests/Documentation/` (files prefixed `Session_` or `Kickstart_`) and will be moved
here in a follow-up PR (see [../SDC.Schema/docs/roadmap.md](../SDC.Schema/docs/roadmap.md)).
